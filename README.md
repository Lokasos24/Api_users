## Api users ##

## Descripción del proyecto
Este proyecto muestra el consumo de una API externa, mostrando los datos en el DOM, incluyendo separación de responsabilidades.
Siguiendo una estructura de carpetas para orden.
/api_users
  /scripts
    /dom
      /event.js
    /services
      /createCards.js
      /renderUI.js
    /utils
    main.js
  index.html
  styles.css

## Principales funcionalidades
- Muestra cartas en el HTML ✔️

## Funcionalidades adicionales
- Filtrar usuarios por condiciones ✔️
- Mostrar en tarjetas visuales ✔️
- Agregar buscador ✔️

## Principales aprendizajes
Principalmente lo que aprendi es: 
- Como consumir una API con fetch/async-await.
- Como manejar estado local y renderizar dinamicamente.
- Estructura de carpetas modular.
- Renderizar UI cuando se pasan datos cambiados.

## Instalación
- Clonar el repositorio.
- Abrir index.html en tu navegador

## Tecnologías
- JavaScript
- CSS
- HTML
